# 嵌入式调试基础设施 — AI 提示词

版本：v1  
来源：MSPM0G3507 双轴云台 I2C 四重 bug 6 小时调试实战

---

## 场景

嵌入式项目代码跑起来了但行为不对，没有 printf/串口输出，全靠 JLink/调试器看内存。让 AI 帮你搭建完整的调试基础设施。

---

## 快速使用

```
我这是一个 [MCU 型号] 的 [项目描述]，[传感器] 通过 [接口] 读取，
[执行器] 用 [驱动方式]。现在 [问题现象]，
帮我搭建完整的调试基础设施，
包括应用层 DebugSnapshot、驱动层调试结构体、心跳 GPIO，
并指导我如何在调试器中查看这些变量。
```
